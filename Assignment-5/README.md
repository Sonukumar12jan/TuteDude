# TuteDude_Assignment---05

## Task - 01
A dictionary stores data in the form of key–value pairs, where each key is unique.
Here, student names are used as keys and their marks are stored as values, which makes searching and retrieval very fast.
The input() function allows the user to enter a student’s name dynamically at runtime.
The in keyword checks whether the entered name exists in the dictionary.
If the student name is found, the corresponding marks are retrieved using the key and displayed.
If the student name does not exist, the program displays a meaningful message instead of causing an error.

## Task - 02
The list is created using range(1, 11), which generates numbers from 1 to 10.
The list() function converts the generated range into a list format.
List slicing [:5] extracts elements starting from index 0 up to index 4, giving the first five elements.
Reverse slicing [::-1] reverses the extracted list without using loops or additional logic.
Printing the original list, sliced list, and reversed list helps clearly understand how slicing works.
