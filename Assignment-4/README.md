# TuteDude_Assignment---04
## Task 1
1. The program starts by printing a heading message.
2. It attempts to open the file sample.txt in read mode using a try block.
3. If the file exists:
    * Each line is read one by one
    * Line numbers are added while printing
4. If the file does not exist:
    * The FileNotFoundError is caught
    * A user-friendly error message is displayed instead of stopping the program
  
## Task 2
1. The program asks the user to enter text.
2. The entered text is written to output.txt (write mode).
3. The program then asks for additional text.
4. The additional text is appended to the same file (append mode).
5. Finally, the file is opened in read mode and its complete content is displayed.
