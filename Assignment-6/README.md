# TuteDude_Assignment---06

### Code Functionality Explanation

Tkinter Window Setup
  * A main window of size 500x500 is created using Tk().

Entry Widget
  * Used as a display screen to show user input and results.

Button Click Handling
  * Numeric buttons (0–9) append values to the entry field.

Arithmetic Operations
  * Each operator (+, -, *, /) stores the first number and selected operation.

Equal Function
  * Performs the calculation based on the selected operator and displays the result.

Clear Function
  * Clears the entry field to allow fresh input.
